# PWA Icons

You need to create the following icon files for the PWA:

## Required Icons

- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels

## Design Guidelines

- Use a simple, recognizable design
- Ensure good contrast for visibility
- Test on both light and dark backgrounds
- Use a square canvas (1:1 aspect ratio)
- Export as PNG with transparency

## Recommended Design

A simple camera or photo icon with "R" letter or family-related imagery.

## Tools

You can create these icons using:
- [Figma](https://figma.com) (free)
- [Canva](https://canva.com) (free)
- [Photopea](https://photopea.com) (free online Photoshop alternative)
- Adobe Illustrator / Photoshop

## Quick Placeholder

For development, you can use a solid color square with text:
- Background: #3B82F6 (blue)
- Text: "R" in white, centered
- Font: Arial Bold, 120px for 192px icon, 320px for 512px icon

## Generation Tools

You can also use online PWA icon generators:
- [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)

Just upload a base image (512x512) and it will generate all required sizes.
